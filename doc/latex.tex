\subsection{Bullet Points}

Bulletpoints are coded using the \texttt{itemize} environment, and are usefull for listing the entries of a set.

\begin{latexdemo}
  \begin{minted}[breaklines]{latex}
  \begin{itemize}
    \item Bonobo
    \item Gorilla
    \item Gibbon
  \end{itemize}
  \end{minted}
  &
  \begin{itemize}
    \item Bonobo
    \item Gorilla
    \item Gibbon
  \end{itemize}
\end{latexdemo}
